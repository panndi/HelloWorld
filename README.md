# HelloWorld WinForms Application

This is a simple Windows Forms (WinForms) application built with C# and the latest .NET SDK.

## How to Build and Run

1. Press `Ctrl+Shift+B` to build the project.
2. Press `F5` to run the application in debug mode.

Alternatively, use the following commands in the terminal:

```
dotnet build
dotnet run
```

## Project Structure
- `HelloWorld.csproj`: Project file
- `Program.cs`: Main entry point
- `Form1.cs`: Main form
